接口定义 #

作用

将对象写入到JSON文件中。

  • 和 writeJson 功能几乎一样，除了当目录不存在时，outputJson会创建该目录

📔：

js
outputJson(file, object[, options][, callback])

// 别名
outputJSON(file, object[, options][, callback])

参数：

  • file <String>

  • object <Object>

  • options <Object>

    • spaces <Number> | <String> 用于缩进的空格数量；或用于缩进的字符串（比如，传入 \t 表示tab缩进）。the space argument
    • EOL <String> 设置EOL（行终止符）字符。默认是 \n
    • replacer - JSON replacer
    • 也接受 fs.writeFile() 配置项

  • callback <Function>

    • err <Error>

示例 #

js
const fs = require('fs-extra')

const file = '/tmp/this/path/does/not/exist/file.json'

// 1️⃣ 使用回调
fs.outputJson(file, { name: 'JP' }, err => {
  console.error(err) // null 😎

  fs.readJson(file, (err, data) => {
    if (err) console.error(err)
    console.log(data.name) // 'JP'
  })
})

// 2️⃣ 使用Promise
fs.outputJson(file, { name: 'JP' })
  .then(() => fs.readJson(file))
  .then(data => console.log(data.name)) // 'JP'
  .catch(err => console.error(err))

// 3️⃣ 使用async/await
aysnc function example() {
  try {
    await fs.outputJson(file, { name: 'JP' })
    const data = await fs.readJson(file)
    console.log(data.name) // 'JP'
  } catch (err) {
    console.error(err)
  }
}
